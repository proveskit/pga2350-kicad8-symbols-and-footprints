# Symbol and footprint library for KiCad 8

[A small collection of KiCad 8 symbols and footprints](https://github.com/vurdalakov/kicad8-symbols-and-footprints) I've created due to their unavailability elsewhere.

Distributed under the terms of the [MIT license](https://opensource.org/licenses/MIT).

## Pimoroni PGA2350

[Pimoroni PGA2350](https://pimoroni.com/pga2350) is a minimal but powerful [RP2350](https://en.wikipedia.org/wiki/RP2350) breakout board modelled on a [Pin Grid Array](https://en.wikipedia.org/wiki/Pin_grid_array), with the maximum exposed pins crammed into the smallest possible space.

* Home page: https://shop.pimoroni.com/products/pga2350
* [Pinout diagram (printable PDF)](/Pimoroni_PGA2350/docs/pimoroni_pga2350_pinout_diagram.pdf)
* [Schematic diagram (printable PDF)](/Pimoroni_PGA2350/docs/pimoroni_pga2350_schematic_diagram.pdf)
